# maha

Mahalakshmi Home Pickles website with online ordering support.

## New order options

- **WhatsApp order**: customers can send order details directly to the shop WhatsApp number from the top order form.
- **Razorpay checkout**: customers select products in the catalog, open the cart drawer, and complete payment securely via Razorpay.

## Notes

- The top order form is now wired for WhatsApp order requests.
- The cart drawer handles the Razorpay payment flow.
- Environment configuration for Razorpay and WhatsApp notifications is documented in `RENDER_ENV_SETUP.md`.
